## 金策 FinSight —— 可信金融投研与财富决策智能体平台

**一句话**：基于多智能体、可信证据链、因果推演、组合分析和合规审查，把分散的金融信息转化为可追溯、可解释、可审核、可执行的投研结论和财富管理方案。

**不是什么**：不是财经聊天机器人、不是新闻摘要工具、不是自动荐股系统、不替代持牌投顾。AI 仅辅助研究与决策，重要建议保留人工确认，全程审计留痕。

**状态**：方案设计 + 产品原型阶段（入围后开发 MVP）。

### 核心创新
1. **证据原生**：数据—证据—推理—结论—建议全链路，而非生成后补引用
2. **事实/观点/推断分层**：市场观点不会被包装成已确认事实
3. **因果推演 + 失效条件**：呈现传导路径、关键假设、反例与结论失效条件
4. **投研→财富一体化**：研究结论直接映射到客户组合、风险画像与沟通材料
5. **合规即流程**：适当性、风险提示、证据审核、人工审批内嵌工作流
6. **多智能体交叉审核**：11类角色分工（规划/数据/宏观/基本面/因果/情景/组合/财富/合规/证据审计/整合）
7. **人机协同**：人工判断与责任边界清晰

### 旗舰场景
"美联储加息信号 → 分析对黄金、原油、美股、人民币资产及某模拟客户组合的影响 → 输出内部投研报告 + 客户沟通材料，全程证据链与合规留痕。"
