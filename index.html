01 / PROBLEM

行业痛点：不是缺信息，而是缺可信结论

核心矛盾已从信息稀缺转为"可信结论稀缺"——终端解决"查得到"，没人解决"信得过、用得上、审得过"。

信息分散在行情、公告、财报、政策、研报与内部资料中；通用大模型存在幻觉、数字错误与引用缺失；Bloomberg、Wind Alice、AlphaSense 等头部产品仍停留在搜索与摘要，缺少从宏观事件到客户组合的决策闭环；投研、财富、合规流程割裂，结论难溯源、数字不敢引用、合规事后补漏。与此同时，金融大模型已进入规模化落地（同花顺 HithinkGPT 成为业内首个通过网信办备案的金融对话大模型，2025年2月至少16家券商完成 DeepSeek 本地化部署），合规是前置约束而非事后补丁。

02 / SOLUTION

解决方案：六层架构的人机协同决策流水线

不是财经聊天机器人，不是荐股系统——AI 仅辅助研究与决策，重要建议保留人工确认，全程审计留痕。

L1

多源数据与知识层

行情/宏观/公告/财报/政策/研报/内部资料统一接入，来源标记、时间有效性、权限分级、敏感脱敏。

L2

多智能体投研引擎

规划/数据/宏观/基本面/因果/情景/组合/财富/合规/证据审计/整合，11类角色交叉审核。

L3

可信证据链

每项结论下钻至原文位置，附假设、反例、置信度、失效条件；事实/观点/推断分层。

L4

因果推演

回答事件沿何路径、在什么条件下、在多长尺度上影响哪些资产，附结论失效条件。

L5

组合与财富模块

研究结论映射到客户持仓、风险等级、期限与流动性约束，多方案比较，双版本报告。

L6

合规即流程

适当性、收益承诺检测、风险提示完整性、人工审批内嵌工作流；全程留痕，支持私有化部署。

03 / FLAGSHIP SCENARIO

旗舰场景：从宏观事件到客户组合建议的完整闭环

"美联储加息信号 → 黄金/原油/美股/人民币资产 + 模拟客户组合 → 内部投研报告 + 客户沟通材料"。

检索央行讲话与权威数据 → 事实/观点/推断分层 → 构建利率—美元—实际利率—黄金—权益—汇率因果链 → 乐观/基准/悲观三情景 → 模拟客户组合压力测试 → 按风险等级与期限约束生成备选方案 → 合规审查 → 人工确认 → 双版本报告输出 → 证据链与审计归档。

演示页面与样本中的客户、持仓与市场数值均为模拟数据，仅用于展示产品逻辑，不构成投资建议。

06 / INNOVATION

核心创新

创新不在大模型、RAG、多智能体本身，而在以下七点。

I-1

证据原生

数据—证据—推理—结论—建议全链路形成于研究过程，而非生成后补引用。

I-2

分层表达

事实观点推断界面级区分，市场观点不会被包装成已确认事实。

I-3

因果附失效条件

不只给方向，还给传导路径、关键假设、反例与结论失效条件。

I-4

投研财富一体化

研究结论直接映射到客户组合、风险画像与沟通材料。

I-5

合规即流程

适当性、风险提示、证据审核与人工审批内嵌于工作流，而非事后补漏。

I-6

交叉审核 + 人机协同

11类角色互相校验降低单模型错误；重要结论保留人工判断与责任边界。
