# 金策 FinSight 数据分析与报告样本（II）

> 场景统一为贯穿 Demo："周一 9:05 央行宣布降准 0.5 个百分点 → 某区域财富工作室 6 位顾问、400 位客户（其中 180 位持有固收+产品）、在售 8 只产品 → 开盘后 1 小时内完成产品影响判断、客户 4 组分层与分组话术审批发送"。
> 当前状态：以下内容是字段、写法和流程的演示样稿，不是一次已经完成的市场分析或合规审批。全部客户、持仓、产品与数据均为**模拟数据**，不对应任何真实机构、产品或个人；`T0`、`X.X`、比例与人数均为模拟/示例输入。正式运行时须先锁定证据截止时间，再由持牌顾问确认。全文不构成投资建议。
> 分工说明：话术生成与合规审核环节的样稿见 `samples/话术样本.md`；本文件覆盖话术之前的四个环节——资讯解读、观点整理、产品影响与客户分层，外加证据卡与失效重审两个机制样例。

## 样本 1：资讯解读卡

**演示样本·基于公开信息与模拟数据生成·非投资建议**

### 降准公告解读（资讯研报库记录 V-ZX001·模拟数据）

#### 一、事件概述

本稿处理一个假设场景：周一 9:05（记为 `T0`），央行官网发布公告，宣布下调金融机构存款准备金率 0.5 个百分点。样稿未指定真实公告日期与原文版本，引号内措辞不得当作真实引文。正式分析的第一步是锁定 `T0`、下载公告原文并记录证据截止时间。

#### 二、事实清单（带证据引用）

| 编号 | 事实陈述 | 证据 | 分层 |
|---|---|---|---|
| F1 | 央行宣布下调金融机构存款准备金率 0.5 个百分点（示例表述） | [E1] | 待确认：缺公告日期与原文版本 |
| F2 | 公告说明本次降准释放长期流动性约 X 万亿元（示例数值） | [E1] | 待确认：需核对公告原文口径 |
| F3 | 公告提及"保持流动性合理充裕"（示例概括） | [E1] | 无法确认：须区分逐字原话与媒体概括 |
| F4 | 公告后债市现券收益率下行约 X bp（示例数值） | [E3] | 待确认：依赖行情快照时点与券种口径 |
| F5 | 部分公开评论认为后续可能接续其他宽松工具 | [E5] | 观点汇总，不作为事实 |

证据登记（节选，完整证据卡格式见样本 5）：
- [E1] 央行官网降准公告，示例证据时间 `T0`；正式记录需含发布日期、版本与原文锚点
- [E3] 公开债市行情快照，示例快照时间 `T0+1h`；盘中数据不能替代收盘数据
- [E5] 公开新闻与机构评论，示例检索截止时间 `T0+1d`；仅用于标记观点分歧

#### 三、分层标注

- **已确认事实**：暂缺——F1–F4 均为待确认状态，正式系统下须完成 [E1][E3] 原文核对后方可升级；
- **市场共识**：降准释放流动性、方向上利好债券（[E5]，需标注为共识而非事实）；
- **模型推断**：宽松预期若已充分定价，落地后反应可能有限（推断，证据不足，不进客户话术）。

## 样本 2：观点条目

**演示样本·基于公开信息与模拟数据生成·非投资建议**

### 观点库记录 G01（模拟数据，字段与 `docs/08_产品与客户数据模型.md` §三一致）

| 字段 | 内容 |
|---|---|
| 结论 | 降准释放流动性，市场利率存在下行动力，对以债券为底层的产品偏正面 |
| 立场 | 利多（债券类底层） |
| 适用情景 | 经济基本面平稳、通胀未超预期反弹、宽松未被提前充分定价 |
| 关键假设 | 银行体系将释放资金投向债券等资产；无同期外部冲击覆盖利率影响 |
| 反例 | 若宽松已被充分定价，落地可能"利好出尽"，收益率不降反升 |
| 失效条件 | 通胀数据连续超预期；央行后续表态转向中性；降准落地后收益率不降反升 |
| 失效状态 | 有效（示例状态） |
| 最后重审时间 | `T0+2h`（示例） |
| 引用支持判定 | 部分支持——[E1] 支持降准事实与流动性释放，"对债券偏正面"为机制推断 |
| 证据锚点 | 央行官网公告，`T0`，原文第 1 段（示例锚点） |
| 观点归属机构 | 本机构（模拟） |

> 说明：失效状态、最后重审时间、引用支持判定三项由自研外挂写入（`docs/07_飞书实现方案.md` §3.2 外挂二、三，当前未实现，此处为设计样例）。

## 样本 4：客户分层结果

**演示样本·基于公开信息与模拟数据生成·非投资建议**

### 400 位客户 → 4 组（模拟数据）

**分层规则**：风险等级（稳健 R2–R3 / 积极 R4–R5）× 是否持有固收+产品（以持仓明细类别字段判定，如"固收+为主"）。分层输入字段全部来自脱敏后的客户库（`docs/08` §五），不含任何身份信息。

| 组别 | 风险等级 | 是否持有固收+ | 组内人数（模拟） | 分组依据 |
|---|---|---|---|---|
| 第 1 组 | R2–R3 | 持有 | 140 | 风险等级 R2–R3 且持仓类别含固收+ |
| 第 2 组 | R4–R5 | 持有 | 40 | 风险等级 R4–R5 且持仓类别含固收+ |
| 第 3 组 | R2–R3 | 未持有 | 150 | 风险等级 R2–R3 且持仓类别不含固收+ |

## 样本 5：证据卡

**演示样本·基于公开信息与模拟数据生成·非投资建议**

### 证据卡：一条结论的完整下钻

**待审命题**：降准落地后，以债券为主要底层的产品净值短期内偏正面（条件性推断；当前证据未补齐，暂不评级）。

| 字段 | 内容 |
|---|---|
| 证据编号 | [E6] |
| 来源 | 央行官网降准公告（示例来源） |
| 发布时间 | 示例时点 `T0`（正式系统记录精确到分钟） |
| 更新时间 | 示例时点 `T0+1h`（首轮行情快照后复核） |
| 原文位置 | 公告第 1 段"下调金融机构存款准备金率 0.5 个百分点"（正式系统含快照与段落锚点） |
| 事实依据 | 降准 0.5 个百分点、释放长期流动性约 X 万亿元（示例数值）；"流动性释放→利率下行→债券走强"为机制推断，当前没有事件窗口内的完整行情序列 |
| 推断摘要 | 若释放资金投向债券市场且无同期冲击，利率下行动力存在；幅度与持续性无法由单条公告推出 |
| 关键假设 | 通胀未超预期反弹；宽松未被提前充分定价；无重大外部事件覆盖 |
| 反例 | 历史上存在宽松落地后"利好出尽"、收益率回升的阶段（预期提前定价时） |
| 置信度 | 待评定；需补齐事件窗口行情、历史参照期与反方证据 |
| 失效条件 | 通胀数据连续超预期；央行后续表态转向中性；落地后收益率不降反升 |
| 关联证据 | [E1]（公告原文）、[E3]（债市行情快照）、[E5]（机构评论） |
| 审计信息 | 拟由观点整理角色生成、外挂一与外挂三校验；当前均未实际运行，人工确认未开始，提示词版本与模型调用记录尚无可归档实例 |

## 样本 6：失效重审记录

**演示样本·基于公开信息与模拟数据生成·非投资建议**

### 观点 G01 的一次到期重审（外挂二设计样例，当前未实现）

**触发背景**：`T0+5d`（示例时点），最新一期通胀数据公布，同比涨幅超出市场一致预期（模拟数据）。

| 字段 | 内容 |
|---|---|
| 触发事件 | 资讯研报库新增记录 V-ZX014（通胀数据，模拟），命中 G01 失效条件第 1 条"通胀数据连续超预期" |
| 命中规则 | 外挂二失效条件巡检：观点失效条件 ↔ 新入库数据的匹配（待实测确认） |
| 处置动作 | G01 失效状态由"有效"改写为"待重审"；最后重审时间更新为 `T0+5d`；通知责任顾问 |
| 连锁检查 | 绑定 G01 的产品影响卡（P01/P02/P03/P08）与相关话术草稿一并标记"依据待重审"，暂停送审 |
| 重审结论（示例） | 顾问复核后判断：单期通胀超预期尚不构成趋势，G01 维持"有效"，但在适用情景中补充"需跟踪下一期通胀数据"；若结论被推翻，正确处置是**删除或降级该表述**，而非生成一段读起来更像答案的文字 |
| 留痕 | 触发记录、匹配依据、状态变更、顾问处置与时间全部写入观点库对应字段 |

> 说明：本样本演示的是"数据更新后回头重审历史结论"的机制——飞书原生触发器只做"记录变更→执行一次"，不具备这一能力，因此该环节由外挂二承担（判定依据见 `docs/07_飞书实现方案.md` §3.1）。外挂二当前未实现，本记录为设计样例。

## 附：样本与评测的对应关系

- 样本 1 对应指标：事实准确率、数字准确率、信息时效性、表达分层；
- 样本 2/5 对应指标：引用覆盖率、引用支持率、无依据结论比例（证据卡与观点字段完整性）；
- 样本 3 对应指标：产品要素抽取准确率；
- 样本 4 对应指标：客户分层正确率；
- 样本 6 对应指标：失效重审触发召回率与无误触发率（`docs/07` §3.2 外挂二的验证口径）；
- 全部样本生成过程对应指标：单条资讯到全部客户触达的时长、失败率（评测方法见 `docs/06_评测方案.md`）。
